Game Designers

They need to have several soft skills and hard skills; a good Game Designer should be able communicate his/her ideas effectively (Technical Writing) and be able to influence and motivate a team (Storyboarding). Designers often draft early visualization of game features before Coders start the final creation process. A huge part of a designer is to align everyone on the vision for the product. On the technical side, being skilled in graphics programming such as Adobe (Photoshop, Adobe Xd...) help for a better visualization and presenting ideas. Scripting language like LUA is also important for things like designing levels and events.
